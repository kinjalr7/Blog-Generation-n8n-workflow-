# AI Social Media & Blog Post Generator (n8n Workflow)

This repository contains an automated n8n workflow designed to transform a simple blog title into a full content suite. It generates a professional blog article, LinkedIn posts, Twitter threads, and Instagram captions, all while performing an automated SEO audit.

## 🚀 Features
- **Multi-Platform Content:** Generates content for Blog, LinkedIn, Twitter/X, and Instagram in one run.
- **AI-Powered SEO:** Analyzes the generated blog for focus keywords, LSI keywords, and readability scores.
- **Human-in-the-Loop:** Sends a preview to **Telegram** for approval/rejection before final distribution.
- **Multi-Cloud Storage:** Automatically logs data to **Google Sheets** and saves drafts to **Google Drive**.
- **Email Delivery:** Sends a beautifully formatted HTML email via **Gmail** once approved.

## 🛠️ Tech Stack
- **n8n:** Workflow automation.
- **Groq AI:** Powered by `llama-3.3-70b-versatile` for high-speed content generation.
- **Google Workspace:** Sheets, Drive, and Gmail integrations.
- **Telegram:** For real-time mobile notifications and approval triggers.

## 📦 Installation

1. **Prerequisites:** - A running instance of n8n.
   - API Keys for Groq, Google Cloud (OAuth), and a Telegram Bot token.
2. **Importing the Workflow:**
   - Download the `BGEN.json` file from this repo.
   - In n8n, click **Workflows** > **Import from File** and select the JSON.
3. **Configure Credentials:**
   - Set up your **Groq API** credentials.
   - Authenticate your **Google Sheets, Drive, and Gmail** accounts.
   - Add your **Telegram Bot** credentials and update the `Chat ID` in the Telegram node.

## 📋 Usage
1. Open the n8n Form trigger URL.
2. Input your Blog Title, preferred Tone, Language, and Length.
3. Check your Telegram for the preview.
4. Click **✅ Approve** to save to Drive and send the email, or **❌ Reject** to stop the process.

## ⚠️ Security Note
Please ensure you replace the following placeholders in the JSON with your own specific IDs:
- Google Sheet Document ID
- Google Drive Folder ID
- Telegram Chat ID
- Webhook IDs
